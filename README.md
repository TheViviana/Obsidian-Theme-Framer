# Framer Theme

A clean, polished Obsidian theme inspired by [Framer](https://www.framer.com)'s design system.

## Features

- **Dark mode**: Hot pink accents on black canvas
- **Light mode**: Rose gold accents on warm cream background
- **Accent variants**: Built-in classes for `accent-blue`, `accent-cyan`, `accent-purple`
- **Floating status bar**: Slides in from the right on hover
- **Framer-style UI**: Patterned after Framer's sidebar, panels, and component browser
- **WCAG-friendly**: Carefully tuned contrast ratios in both appearances

## Installation

1. Open Obsidian → Settings → Appearance → Themes → Manage
2. Click **Browse** and search for "Framer"
3. Click **Install**

Or manually copy the `Framer/` folder into your vault's `.obsidian/themes/` directory.
